Detailed Description

Time structure.

Time is stored and in both 24-hour and 12-hour modes, and is updated when ds3231_get_time is called.

When setting time and alarms, 24-hour time is used.

If you run your clock in 12-hour mode:

  • set time hour to store in twelveHour and set am to true or false;
  • call ds3231_12h_translate() (this will put the correct value in hour);
  • call ds3231_set_alarm() or ds3231_set_time().

Note that ds3231_set_time(), ds3231_set_alarm_s(), ds3231_get_time_s() and ds3231_get_alarm_s() always operate in 24-hour mode and translation has to be done manually (you can call ds3231_24h_to_12h to perform this calculation).
